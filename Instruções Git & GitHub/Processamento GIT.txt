Os arquivos são encriptografados pelo SHA.

command gitbash
openssl sha1 arquivo


Os objetos GIT
BLOBS, TREE, COMMITS são formas de encapsulamente, seguro por sha1

BLOBS - onde e como os arquivos são guardados

	'blob 9\0conteudo'
	key sha1
metadados
blob - tipo do objeto
9 - tamanhado do arquivo
conteudo - conteudo do arquivo

TREES - armazenam blobs ou outras trees
	sha1

		tree
	README(blob) file(blob) lib(tree)
				simplegit.rb(blob)

COMMITS - é o hash de toda a informação
	tree, parente, autor, conteudo, timestamp

-git commit : "envelopar" o arquivo ( sha1)
-git status : monitorar o status dos arquivos
